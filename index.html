📊 9:05 量比訊號

計算方式

當天第一根 5 分鐘 K 棒成交量 ÷ 前一交易日全日成交量 × 100%

為什麼是 15%？能走出波段的股票，開盤第一根量幾乎不低於 12%，超過 15% 代表多空開盤就先決出勝負。

🟢 有呼吸（量比 >15% + 紅K）
📌
進場：量比達標，開盤價附近追進
🛑
停損：跌破開盤價立即出
🎯
停利：前高或 3~5% 先減半，剩餘讓利潤奔跑
🔴 倒貨警報（量比 >15% + 綠K）
⚠️
有人在倒貨，尤其綠K帶長上影線最危險
📌
手上有單：第一聲警報，立即評估是否減碼
📌
做空：等 9:30 確認跌破開盤價後行動

📈 均線趨勢（MA5 / MA20 / MA60）

用途：確認中長線方向，避免逆勢操作
🟢
多頭排列：MA5 > MA20 > MA60，趨勢向上，做多訊號更可靠
🔴
空頭排列：MA5 < MA20 < MA60，趨勢向下，即使量比有呼吸也要小心，可能只是反彈
🌟
黃金交叉：MA5 剛上穿 MA20，趨勢轉多訊號，配合量比有呼吸是最強組合
💀
死亡交叉：MA5 剛下穿 MA20，趨勢轉空，要特別謹慎
🌟 最強訊號（三合一）

量比 >15% + 紅K（有呼吸）+ 黃金交叉 + 多頭排列 → 四個條件同時成立，這是最高優先級的進場機會，勝率最高。
